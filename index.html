SkEditor

Jest aktualnie jedynym wspieranym edytorem specjalnie do skryptów (w języku skript) w polsce.

Program zawiera:

  • Poświetlanie składni

    Praca przy tym, jest owiele przyjemniejsza.

  • Pasek plików

    Dzięki paskowi plików, możesz mieć kilka otwartych plików na raz.

  • Ładne UI

    UI jest piękne jak motyl, przyjemne przy pracy.
